About Project Codiname Husky

Project Codiname Husky is a platform dedicated to centralizing and facilitating access to all Lore Books from the Destiny universe, using the official Bungie API.

Technologies Used:

  • HTML5, CSS3, JavaScript
  • Bungie.net API
  • Fetch API for requests
